本地SQLite存储

轻量级数据库

使用SQLite作为本地数据库，无需服务器，数据完全存储在本地文件中。

数据库结构：

  • • indicators - 存储所有经济指标数据
  • • modules - 7大模块配置
  • • scores - 每日评分记录
  • • metadata - 指标元数据

需要Python后端支持CSV导入和数据库操作。详见项目文档。

Python数据导入脚本

# data_import.py
import pandas as pd
import sqlite3
from pathlib import Path

# 连接到SQLite数据库
conn = sqlite3.connect('macro_data.db')

# 读取CSV并导入
def import_fred_csv(csv_path, series_id):
    df = pd.read_csv(csv_path, skiprows=0)
    df['series_id'] = series_id
    df.to_sql('indicators', conn, if_exists='append', index=False)
    print(f"Imported {series_id}: {len(df)} rows")

# 示例：导入多个指标
indicators = {
    'DGS10': 'data/DGS10.csv',      # 10年期国债收益率
    'UNRATE': 'data/UNRATE.csv',    # 失业率
    'GDPC1': 'data/GDPC1.csv',      # 实际GDP
    'CPIAUCSL': 'data/CPIAUCSL.csv' # CPI
}

for series_id, path in indicators.items():
    import_fred_csv(path, series_id)

conn.close()
print("数据导入完成！")
