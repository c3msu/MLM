从FRED下载数据

无需API密钥

  1. 1 访问 FRED官网
  2. 2 搜索指标代码（如：DGS10, VIXCLS）
  3. 3 点击"Download"按钮，选择CSV格式
  4. 4 保存到 data/ 目录

DGS10 VIXCLS UNRATE FEDFUNDS

本地SQLite存储

轻量级数据库

使用SQLite作为本地数据库，无需服务器，数据完全存储在本地文件中。

数据库结构：

  • • indicators - 存储所有经济指标数据
  • • module_scores - 7大模块评分
  • • overall_scores - 总体评分记录

需要Python支持CSV导入和数据库操作

Python数据导入脚本

# scripts/import_data.py
import pandas as pd
import sqlite3
from pathlib import Path

DB_PATH = 'macro_data.db'
DATA_DIR = Path('data')

def import_csv(csv_file, series_id, module_name):
    df = pd.read_csv(csv_file)
    df['series_id'] = series_id
    df['module'] = module_name
    
    conn = sqlite3.connect(DB_PATH)
    df.to_sql('indicators', conn, if_exists='append', index=False)
    conn.close()
    print(f"导入 {series_id}: {len(df)} 条记录")

# 使用示例
import_csv('data/DGS10.csv', 'DGS10', 'treasury')
